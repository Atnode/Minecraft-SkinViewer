Minecraft SkinViewer est une visionneuse de skins Minecraft.

Vous pouvez ouvrir un skin à partir d'une adresse web (par exemple, un lien direct vers l'image d'un skin) ou à partir d'un pseudo d'utilisateur de Minecraft.

Minecraft SkinViewer est un fork d'un projet open source sous la licence GNU GPLv3, nommé "Skin Manager", disponible sur GitHub et développé par Corbin Davenport.

Minecraft SkinViewer est développé par Atnode.
